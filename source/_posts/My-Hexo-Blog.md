# Hexo and GitHub 
<!--more-->
1. Installation
```
$ npm install hexo-cli -g
```
2. Quick Start
Setup your blog
```
$ hexo init blog
$ cd blog
```
3. Start the server
```
$ hexo server
```
4. Create a new post
```
$ hexo new "Hello Hexo"
```
5. Generate static files
```
$ hexo generate
```
6. use git
```
npm install hexo-deployer-git –save 
```
